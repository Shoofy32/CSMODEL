# CSMODEL

## Git LFS
For file dataset storage please install [Git LFS](https://git-lfs.com/) by opening a terminal within the repo folder and typing the following:    
*Mac via homebrew*     
`brew install git-lfs`    
*Windows*    
`git lfs install`    

The .gitattributes file has already been configured to have LFS track ".csv" files so just install it on your own machine so commits and pushes work as intended.

Also please note Github has a max singular file limit of [2GB](https://docs.github.com/en/repositories/working-with-files/managing-large-files/about-git-large-file-storage) for the free plan so if you wish to add more datasets please ensure they fall under the 2GB limit
